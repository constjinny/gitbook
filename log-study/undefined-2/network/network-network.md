# 1. 네트워크\(Network\)란?

## 1. 네트워크\(Network\) <a id="e6e7"></a>

* Net+work 두 단어의 합성어이다.
* **전자신호**를 통해 **기기 간 데이터 통신**을 하기 위해 만든 **망**을 의미한다.
* 컴퓨터 네트워크에서 데이터 링크를 통해 연결된 Node간의 집합을 뜻함.
* 통신 방법으로는 케이블 등으로 장치를 집접 연결한 유선 네트워크와 물질적인 연결이 없이 전파만으로 통신하는 무선 네트워크가 있다.

## 2. 네트워크의 종류 <a id="ecb5"></a>

![](https://miro.medium.com/max/60/0*zJ3mKsFhS74kfqzo?q=20)![](https://miro.medium.com/max/938/0*zJ3mKsFhS74kfqzo)[규모에 따라 분류해놓은 무선 통신망 비교도.](https://namu.wiki/jump/kil0wdwfyohMANO%2F6ZRg7rQk4xRaIcAiGyM5LIfivdM1GTbICI85k95DzX7eCk35Tr406gYx7lF7gjOV24mvsmt1WoU1iCaCt%2BD%2FXs6rmCFYpQAA8TWTori0oO3Mt18z1h2lBwEipMZZPb6d5%2Bqe4A%3D%3D)

이 외에도 통신망의 종류가 몇가지 더 존재하지만, 나는 5종류 정도로 정리해보았다.

### ✔️ WAN \(Wide Area Network, 광역 통신망\) <a id="1e5d"></a>

* 가장 넓은 범위의 통신망이다.
* 일반 개인이 설치하기 어렵고, 정부나 사업체에서 지역/국가 간 제공하는 네트워크이다.
* ex. ISP\(인터넷 서비스 제공업자\)가 인터넷 서비스를 위해 전국에 회선을 설치해 인터넷을 제공, 전화선 등

### ✔️ MAN \(Metropolitan Area Network, 지역 통신망\) <a id="9712"></a>

* 지역 범위의 통신망이다.
* 주로 광섬유 케이블\(Backbone Cable\)로 지역내의 LAN들을 연결한다.

### ✔️ LAN \(Local Area Network, 근거리 통신망\) <a id="9e05"></a>

* 집, 학교, 회사 등 하나의 층, 건물 정도의 범위를 가진 네트워크이다.
* 서버를 중심으로 다수의 컴퓨터가 연결된다.
* ex. WiFi

### ✔️ PAN \(Personal Area Network, 단거리 통신망\) <a id="3ec9"></a>

* 개인 네트워크로 이해하자.
* ex. 개인의 데스크탑과 주변기기… 블루투스, USB

### ✔️ BAN \(Body area network, 인체 통신망\) <a id="6f73"></a>

* 개인이 인체에 지니고 다닐 수 있는 정도의 네트워크를 의미한다.
* ex. 스마트폰, 애플워치…

## 3. 네트워크 데이터 통신 <a id="55d6"></a>

* 통신 \(Telecommunications\) : 물질적으로 떨어져 있는 상태에서 송신 장치를 통해 정보를 전송하고, 전송 매체를 타고 넘어온 정보를 수신 장치를 이용하여 식별이 가능한 형태로 변경하여 받는 것을 의미한다.
* 데이터 \(Data\) : 검증이 되지 않은 **단순하게 수집된 자료**들을 의미한다.
* 정보 \(Infomation\) : 검증되고 **가공된 데이터**를 의미한다.

## 4. 네트워크 통신규약 === Protocol <a id="5357"></a>

* 다양한 통신 장비사이에서 **데이터를 주고받기 위해 정해진 양식과 규칙**
* 프로토콜을 지키면 운영체제가 다른 컴퓨터간에도 데이터 이동이 가능하다. **즉 프로토콜이 지켜지지 않으면 네트워크 통신을 할 수 없다.**
* 라우터, 서버, 스위치 등 네트워크 통신이 가능한 장치들은 프로토콜 기반으로 데이터를 주고 받는다.
* 프로토콜은 하나가 아닌 네트워크 통신을 위한 **다양한 프로토콜**이 존재한다.
* ex. http, https, ssh …
